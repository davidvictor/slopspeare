# Quick reference

One screen. The tells that matter most, with the instant fix. Full reasoning lives in [`../RULES.md`](../RULES.md).

## The twelve worst tells

| Tell | Fix | Rule |
| --- | --- | --- |
| Em dash inside a sentence | Period, comma, or colon | `PUNCT-1` |
| "X, not Y" / "not just X, but Y" / "isn't X, it's Y" | State the one thing that matters | `RHET-1` |
| delve, tapestry, pivotal, underscore, meticulous, robust, seamless | Plainer word, or rewrite around it | `VOCAB-1`, `VOCAB-2` |
| "In today's fast-paced world…" | Open on the actual point | `RHET-5` |
| "Here's the thing," "Let's dive in" | Start with the substance | `RHET-7` |
| "It's worth noting that," "In conclusion" | Just say the thing | `VOCAB-5` |
| Furthermore / Moreover / Additionally | Cut it, or "also" | `VOCAB-6` |
| "improved performance significantly" | Give the number: "4.2s to 800ms" | `SUBST-1` |
| "the foundation was rebuilt" | "I rebuilt the foundation" | `STRUCT-4` |
| Every paragraph wraps in a tidy bow | Let some thoughts just end | `SUBST-14` |
| Three sentences, same shape, in a row | Vary length and opening | `STRUCT-1` |
| "As an AI…", "I hope this helps!" | Delete entirely | `REVISE-10` |

## The instant rewrite pattern

Most slop is one of three moves. The fix is the same every time.

1. **It performs instead of states.** "This represents a pivotal moment" → say what happened.
2. **It is generic enough to belong to anyone.** "results-driven and user-centered" → name the specific thing only you did.
3. **It hedges instead of committing.** "it could be argued that" → take the position.

## The thirty-second pass

Read the draft out loud. Stop on any line you would never say to a friend. Rewrite that line. Repeat.
